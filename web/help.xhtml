Plan Help Page

Registration Help



Make sure you do not already have an account. If you do you won't be able to register.

Make sure your passwords match identical.

You password needs to contain both a character and number, and be over 8 characters long.

You can click on the register button to go directly back to the login page after registering.

Login Help



You must have already registered with FindAPlan. If you have not done so, please do so prior to logging in!

Make sure your password is typed correctly. Please note that passwords are case sensitive, while usernames are not.

Recall that passwords have specific constraints. You need both a number and a character.

You can create another account through the registration link at the top of the page if you have forgotten your credentials.

Input Help



You should only enter whole numbers for minutes and texts!

Data use is in the form of gigabytes, but you can use decimals.

Please enter input for all fields. If you need no minutes, for example, you can enter 0.

Remember that all data input should be on a monthly basis.

Output Help



You can click on more info to go to the plan providers external website.

The plans are sorted in terms of cost, with the cheapest 'winning' gold.

You can easily click on the back button to return to the input page to enter in new information.

Plans are chosen from almost 40 plans, from the main mobile carriers: AT+T, Boost Mobile, Sprint, T-Mobile and Verizon.

For additional help, please Contact us.
